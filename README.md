# AArch64 QEMU Mac Template
A template for AArch64 development using QEMU on macOS



## How to Use

1. Please follow the [EnvironmentSetup.md](./EnvironmentSetup.md) to setup the environment.

2. Open the workspace `aarch64_qemu.code-workspace` using VSCode.

3. Press CMD + Shift + B to build and run.
